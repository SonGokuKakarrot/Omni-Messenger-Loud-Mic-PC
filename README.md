# Omni Messenger Lord

A Manifest V2 browser extension for Quetta/Chromium-style Android browsers that mirrors the Omni DC Lord control panel and loud microphone profile, but targets Facebook and Messenger browser calls.

## Supported call pages

- `https://facebook.com/*`
- `https://*.facebook.com/*`
- `https://messenger.com/*`
- `https://*.messenger.com/*`

Messenger unlimited volume.
